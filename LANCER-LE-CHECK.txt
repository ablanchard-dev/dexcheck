DEXCHECK - PC CHECK
=================

A faire en partage d'ecran (Discord/AnyDesk) avec un responsable Warzup.
Le check ne modifie rien sur ton PC : il lit et produit un rapport.

LE PLUS SIMPLE (double-clic)
----------------------------
1. Telecharge le dossier (bouton vert "Code" > "Download ZIP" sur GitHub),
   puis clic droit sur le .zip > "Extraire tout".
2. Double-clique sur   LANCER-LE-CHECK.bat
3. Si le moderateur te dicte un mot, tape-le (sinon appuie juste sur Entree).
4. Une fenetre bleue (UAC) demande l'autorisation admin : clique OUI.
   (necessaire pour lire l'historique des suppressions, les journaux...)
5. Le check se deroule a l'ecran. A la fin :
   - un VERDICT (CLEAN / A VERIFIER / SUSPECT / ROUGE),
   - un rapport sur ton Bureau (DexCheck_<PC>_<date>.txt et .html),
   - un code SHA256 affiche : c'est l'empreinte du rapport, note-la / screen.
6. Appuie sur Entree pour fermer.

SI LE DOUBLE-CLIC NE MARCHE PAS
-------------------------------
Ouvre PowerShell (touche Windows, tape "powershell", Entree) et colle :

   powershell -NoProfile -ExecutionPolicy Bypass -File "%USERPROFILE%\Downloads\DexCheck.ps1"

VERSION APPROFONDIE (si un responsable le demande)
--------------------------------------------------
Ajoute -Deep a la fin de la commande PowerShell ci-dessus (plus long).

Si tu refuses l'UAC, le check tourne quand meme mais en mode partiel (certaines
verifications affichent "N/A") - le responsable le verra a l'ecran.
