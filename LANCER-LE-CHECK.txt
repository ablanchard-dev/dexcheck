DEXCHECK - PC CHECK
=================

A faire en partage d'ecran (Discord/AnyDesk) avec un responsable Warzup.
Le check ne modifie rien sur ton PC : il lit et produit un rapport.

METHODE ZERO AVERTISSEMENT (recommandee)
----------------------------------------
1. Telecharge le ZIP (bouton vert "Code" > "Download ZIP" sur GitHub).
2. AVANT d'extraire : clic droit sur le .zip > Proprietes >
   coche "Debloquer" (case en bas a droite) > OK.
   -> ca retire l'etiquette "venu d'internet", donc AUCUN avertissement ensuite.
3. Clic droit sur le .zip > "Extraire tout".
4. Double-clique sur   LANCER-LE-CHECK.bat
5. Si le moderateur te dicte un mot, tape-le (sinon appuie sur Entree).
6. "Mode approfondi ? [o/N]" : tape N (Entree) par defaut. Reponds o UNIQUEMENT
   si un responsable te le demande (check plus complet, plus long).
7. Fenetre bleue (UAC) : clique OUI (pour lire suppressions, journaux...).
8. Le check se deroule. A la fin : un VERDICT, un rapport .txt/.html sur le
   Bureau, et un SHA256 (empreinte du rapport, note-le / screen). Entree pour fermer.

SI TU AS OUBLIE L'ETAPE 2
-------------------------
Au 1er double-clic, Windows peut afficher "Open File - Security Warning" ou
"Windows a protege votre PC". C'est normal (fichier non signe), ce n'est PAS un
virus : clique "Informations complementaires" > "Executer quand meme"
(ou simplement "Executer"). Le launcher se debloque tout seul : aux lancements
suivants, plus aucun avertissement.

SI LE DOUBLE-CLIC NE MARCHE VRAIMENT PAS
----------------------------------------
Ouvre le dossier extrait (celui qui contient DexCheck.ps1), clique dans la barre
d'adresse en haut, tape  powershell  puis Entree. Une fenetre PowerShell s'ouvre
DANS ce dossier. Colle :

   powershell -NoProfile -ExecutionPolicy Bypass -File .\DexCheck.ps1

Pour le mode approfondi, ajoute -Deep a la fin ; pour le mot du modo, ajoute
-Nonce "le-mot". Exemple :  ...\DexCheck.ps1 -Deep -Nonce "panda-47-bleu"

Si tu refuses l'UAC, le check tourne quand meme mais en mode partiel (certaines
verifications affichent "N/A") - le responsable le verra a l'ecran.
